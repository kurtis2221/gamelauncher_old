/============================\
|Game Launcher made by Kurtis|
\============================/

EN:
===
It's a simple program that you can use for launcing games. I think most people hate that
if their folders/programs are open when playing the game. With this program this won't be
a problem anymore, and it has some extras than a windows folder manager.

Pros:
- Simple and easy
- It closes when launching a game
- You can add sounds to it (if you want)
- Low memory useage (max 10MB)

Cons:
- Windows XP or higher needed to run (because .net 3.5)
- Only the main icon can be used as icon
- It don't have a GUI (Graphical User Interface) for editing config, only with notepad
- Limited characters at game names
- When having an error it shows the line number

HU:
===
Egy egyszerû program amivel játékokat indíthatunk el. Úgy gondolom sokan nem szeretik, ha
a mappájuk/programjuk nyitva van amikor játékokkal játszanak. Ezzel a programmal többé
nem lesz ilyen probléma, és még pár extrával bír amivel a windows mappakezelõ nem.

Elõnyök:
- Egyszerû és könnyû
- Bezárja magát amikor elindítja a játékot
- Hangokat adhatsz hozzá (ha akarsz)
- Alacsony memóriahasználat (max 10MB)
- Hiba esetén kidobja a sor számot és hibaüzenet

Hátrányok:
- Windows XP vagy késõbbi kell a futtatáshoz (.net 3.5 miatt)
- Csak a fõikon használható ikonként
- Nincs GUI-ja (Grafikus Felhasználói Felület) a beállítások szerkesztéséhez, csak jegyzettömbbel lehet
- Korlátozott karakterek a játék neveknél
